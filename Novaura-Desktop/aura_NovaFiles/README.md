## 🎮 Library Reference

### **Most Important for Games**

**2D:**
- `arcade` - Recommended for 2D games (simple, powerful)
- `pygame` - Classic framework

**3D:**

**Models:**

- `blender-python` - Professional modeling

- `pillow` - Image creation/manipulation
- `opencv` - Image processing

- `pydub` - Audio processing
